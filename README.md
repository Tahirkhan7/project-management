# Project Management App

Welcome to the Project Management App, a simple To-Do list application built with React JS. This app allows you to organize your projects, add tasks to them, and manage your to-do items efficiently.

## Demo

Check out the live demo: [Project Management App](https://myprojectmanagement.netlify.app)

## Features

- **Project Management:**
  - Add a new project with a title, description, and due date.
  - Delete existing projects.

- **Task Management:**
  - Add tasks to each project.
  - Delete tasks within a project.

## Usage

1. **Add a Project:**
   - Click on the "Add Project" button.
   - Fill in the project details, including title, description, and due date.
   - Click "Add Project" to create a new project.

2. **Delete a Project:**
   - Hover over a project and click on the delete icon to remove the project.

3. **Add Task to a Project:**
   - Click on a project to view its details.
   - In the project details section, add a task by providing a task name.
   - Click "Add Task" to add the task to the project.

4. **Delete Task:**
   - In a project's details, hover over a task and click on the delete icon to remove the task.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/Tahirkhan7/project_management.git
```

2. Navigate to the project directory:

```bash
cd project_management
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
